# 粘连细胞分割功能更新

## 更新日期
2026年7月17日

## 更新内容

针对用户反馈的"细胞粘连严重"问题，新增了**6种经典图像处理分割方法**，无需机器学习，基于ImageJ和学术文献的成熟算法。

### 新增分割方法

1. **距离变换+分水岭** (`watershed_distance`)
   - 🎯 **首选推荐**用于粘连细胞
   - ImageJ Watershed插件的核心算法
   - 适合圆形/椭圆形粘连细胞

2. **梯度+分水岭** (`watershed_gradient`)
   - 利用边界强度信息
   - 适合边界清晰的细胞

3. **H-minima+分水岭** (`hminima_watershed`)
   - 文献推荐方法
   - 抑制过度分割
   - 适合密集培养的细胞

4. **形态学开运算** (`morphological_opening`)
   - 简单直接
   - 适合轻度粘连（细窄连接）

5. **距离+梯度双重markers** (`combined_markers`)
   - 综合方法
   - 稳定性最好
   - 适合作为"保底"方案

6. **Cellpose深度学习** (`cellpose`) - 原有功能保留
   - 需要额外安装
   - 效果最强但速度慢

### 使用方法

#### 方式1：通过Web界面 (推荐)

1. 启动webui：
   ```bash
   python -m membranequant.webui
   # 或双击 run_webui.cmd
   ```

2. 在"细胞分割方法"下拉菜单中选择：
   - **Otsu阈值法** - 默认，适合无粘连
   - **距离变换+分水岭** - 👈 粘连时首选这个
   - **梯度+分水岭** - 边界清晰时用
   - **H-minima+分水岭** - 过度分割时用
   - **形态学开运算** - 轻度粘连用
   - **距离+梯度双重markers** - 综合方案
   - **Cellpose深度学习** - 需安装cellpose

3. 点击"开始分析"

#### 方式2：通过配置文件

编辑 `config.yaml` 或自定义配置文件：

```yaml
segmentation_method: watershed_distance  # 改成你想用的方法

# 可选方法：
# - otsu
# - watershed_distance
# - watershed_gradient  
# - hminima_watershed
# - morphological_opening
# - combined_markers
# - cellpose
```

#### 方式3：通过Python代码

```python
from membranequant.config import Config
from membranequant.segmentation import segment_whole_cells
import numpy as np

# 加载你的图像
green = ...  # 绿色通道 (EGFP)
red = ...    # 红色通道 (DiI)

# 创建配置
cfg = Config(
    segmentation_method="watershed_distance",  # 选择方法
    minimum_cell_area=500,
    maximum_cell_area=50000
)

# 分割
labels, rejected = segment_whole_cells(green, red, cfg)
```

### 推荐的测试流程

```
遇到粘连问题时：

1. 先试：距离变换+分水岭 (watershed_distance)
   └─ 这是ImageJ最常用的方法，对圆形细胞效果很好
   
2. 如果一个细胞被分成多个（过度分割）：
   └─ 改用：H-minima+分水岭 (hminima_watershed)
   
3. 如果边界不准确：
   └─ 试试：距离+梯度双重markers (combined_markers)
   
4. 如果还是不满意且愿意等待：
   └─ 安装并使用 Cellpose
```

### 技术参考

所有新方法都基于成熟的图像处理算法：

- **分水岭算法**：Vincent & Soille (1991) "Watersheds in digital spaces"
- **距离变换**：OpenCV官方教程、ImageJ文档
- **H-minima变换**：
  - Aydin et al. (2016) "Iterative h-minima-based marker-controlled watershed"
  - Robitaille et al. (2022) "Marker-controlled watershed with deep edge emphasis"
- **形态学操作**：Gonzalez & Woods "Digital Image Processing"

详细算法说明见：`粘连细胞分割方法说明.md`

### 常见问题

**Q: 我应该用哪个方法？**

A: 
- 细胞分离良好 → `otsu`（默认）
- 有粘连 → `watershed_distance`（首选）
- 过度分割 → `hminima_watershed`
- 不确定 → `combined_markers`（保底）

**Q: 需要安装什么吗？**

A: 不需要！所有新方法都用现有依赖（scikit-image, scipy）实现。只有Cellpose需要额外安装。

**Q: 速度会变慢吗？**

A: 大部分新方法和Otsu速度相当，比Cellpose快很多。

**Q: 参数需要调整吗？**

A: 通常不需要。所有算法参数都已根据 `minimum_cell_area` 自动调整。如果效果不好，可以试试调整：
- 最小/最大细胞面积
- 换一个方法

**Q: 原来的数据需要重新分析吗？**

A: 不需要，除非你想用新方法获得更好的分割效果。
